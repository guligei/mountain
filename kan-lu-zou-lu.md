# 看路走路



**极度认真投入当下眼前工作，精益求精**

* 人生·工作的成果 = 思维方式_热情_能力；

**几个维度**

* 人们需要什么，市场需要什么
* 技术可以做到什么
* 你能提供什么

**周围人实际需求**

* 实际问题，真实需求，永远是入口

**宏观把握**

* 现在当下的市场潮流，趋势，市场的需求，市场容量，
* 所有的经典书籍看一遍，经典的开源项目，看尽他人做过的路，大脑有一个整体的地图
* 主动去结识行业内的三波人，你需要帮助的年轻人，同龄竞争的人，最牛逼的人

**人，寻找差距，不平等**

* 行业外的人：有真正需求的人，解决他们痛点，解决实际问题，满足真实需求
* 行业内的人：刚入行的人，同龄入行差不多人，牛逼人，

**构建知识体系**

* 持续绘制知识体系图谱
* 不断的看书实践，搞事情，总结，提高效率，创造价值
